## About Begoo

Begoo is a professional AI-powered voice-to-text application for macOS that transforms your speech into text with exceptional accuracy. Built with privacy in mind, featuring local processing with Whisper.cpp and seamless integration with macOS.

### Key Features
- AI-powered transcription with Whisper.cpp
- Privacy-first local processing
- Multiple transcription engines (Apple, Whisper, Whisper Stream)
- Native macOS design
- Seamless text injection
- Supports macOS 14.0+
